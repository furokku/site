ugoserver

wat this: this is a reimplementation of the flipnote hatena service with a server written in go from scratch.
it's mostly a pet project, but i've been slowly working on adding functionality to it over the past few months.
i do not plan to rival against any other hatena replacements like sudomemo or that new one called reximemo, i am mostly developing this because i find the inner workings of flipnote studio rather interesting. whoever wants to may host it themselves provided you can figure out how, there's like no documentation other than in the source code lmao

how: you can join by setting your dns to 141.147.254.59 and the secondary to whatever works, like 1.1.1.1 (this is recommended, the ds will not be able to connect to anything other than my server if you do not set the secondary dns)
play around with it and stuff idk, it'll help if you report bugs and whatever
